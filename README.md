# Lab Architecture and Operations

The lab environment simulates a small enterprise infrastructure combining on premises services with cloud identity and endpoint management.

Core infrastructure services are hosted on **Windows Server Datacenter 2022 using Hyper V virtualization**.

The environment provides:

Active Directory Domain Services  
Domain Name System  
Dynamic Host Configuration Protocol  

Windows 11 client systems are joined to the domain to simulate enterprise workstation environments used to test authentication workflows, endpoint management, and troubleshooting scenarios.

Cloud services are integrated through **Microsoft 365 and Microsoft Entra ID**, enabling identity management, security policies, and collaboration services.

Endpoint management is performed through **Microsoft Intune**, allowing devices to be enrolled, configured, and monitored through centralized policy management.
